__**Jsoner.txt**__

**Functions:**
Convert minecraft color codes/format codes into a json format, also supports click and hover events. 

**Usage:** 
Chat Colors:
```&0 Black         &1 Dark Blue
&2 Dark Green    &3 Dark Aqua
&4 Dark Red      &5 Dark Purple
&6 Gold          &7 Gray
&8 Dark Gray     &9 Blue
&a Green         &b Aqua
&c Red           &d Purple
&e Yellow        &f White```
**Chat Format:**
&l **Bold**
&o *Italics*
&m ~~Strikethrough~~
&n __Underline__
&k Obfuscated
&r Reset

**Special Format Function:**
```&U Open URL &U         Clicking opens a link
&C Run Command &C      Clicking runs a command
&S Suggest Command&S   Clicking types a command in chat
&T Show Text &T        Hovering shows a text (Tooltips can be colored using & code)
&I Show Item &I        Hovering shows an item 
&E Show Entity &E      Hovering shows an entity
&V Insertion &V        Shift clicking inserts text to chat
&O Score &O            Replaced with the score of a player
&P Selector &P         Replaced with the selector name
&K Keybind &K          Replaced with the key bind name```
__**Remember to have the special codes on both side, to enclose the input for that function.**__

**Examples:**
Shows `World` when hovering on "Hello":```&6&lHello&T&a&lWorld&T```
Execute `/tp @p ~ ~20 ~` command when clicking "Teleport", shows `Up & Up` when hovering above "Up":```&6&lTeleport&C/tp @p ~ ~20 ~&C &e&lUp&TUp && Up&T```
Shows a stone item when hovering above "Stone":```&7Stone&I\"id\":\"stone\",\"Count\":1&I```
Shows a box with the text below:  
```Moo 
Type: Cow``` ```&lMilk&Etype:Cow,name:Moo&E```
Shows "Hello NAME" with colors:```&c&lHello &e&l&P@p&P```
Inserts ":)" when shift clicking `Click to insert`:```&5&lClick to insert&V:)&V```
Clicking on `spawn` runs "/spawn", and clicking `warp` gives you $$w list, and runs "/warp <value>" :```&9&lTeleport to &b&lspawn&C/spawn&C &a&lwarp&C/warp $$w&C```
Shows the key bind for `key.jump`,usually "space": ```C&e&lJump is &a&l&Kkey.jump&K```
